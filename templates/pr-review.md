# PR Review Template

## 목적

이 템플릿은 Pull Request 또는 코드 diff를 리뷰할 때 사용한다.  
리뷰는 단순 스타일 지적보다 실제 장애, 버그, 보안, 유지보수 문제를 우선한다.

---

## 리뷰 대상

- PR 링크:
- 브랜치명:
- 작업 목적:
- 주요 변경 파일:
- 관련 이슈/티켓:

---

## 리뷰 요청 내용

다음 기준으로 코드 변경사항을 리뷰해라.

1. 요구사항을 정확히 만족하는가
2. 기존 기능을 깨뜨릴 가능성은 없는가
3. 예외처리가 충분한가
4. 인증/권한 문제가 없는가
5. DB 정합성 문제가 없는가
6. 성능 문제가 생길 가능성은 없는가
7. 타입 안정성이 지켜졌는가
8. 중복 코드나 책임 분리 문제가 있는가
9. 테스트가 필요한 부분이 빠지지 않았는가
10. 배포 시 위험 요소가 있는가

---

## 중점 리뷰 영역

### 1. 기능 정확성

- 요구사항과 실제 구현이 일치하는가
- 누락된 케이스는 없는가
- 기존 API 응답 형식이 깨지지 않았는가
- 프론트/외부 연동에 영향이 없는가

### 2. 예외처리

- 필수값 누락 처리
- 잘못된 타입 처리
- 존재하지 않는 데이터 처리
- 외부 API 실패 처리
- DB 저장/조회 실패 처리
- null/undefined 처리

### 3. 인증/권한

- 인증이 필요한 API에 guard가 적용되어 있는가
- 관리자/일반 사용자 권한이 분리되어 있는가
- 본인 리소스만 접근 가능한가
- 민감정보가 응답에 포함되지 않는가

### 4. DB/트랜잭션

- 데이터 정합성이 깨질 가능성은 없는가
- 트랜잭션이 필요한 로직은 아닌가
- N+1 쿼리 문제가 없는가
- 인덱스가 필요한 조회 조건은 아닌가
- soft delete 조건이 누락되지 않았는가
- migration이 필요한 변경인지 확인했는가

### 5. 성능

- 불필요한 반복 쿼리는 없는가
- 대량 데이터 처리 시 문제가 없는가
- 페이지네이션이 필요한 조회는 아닌가
- 캐싱이 필요한 부분은 아닌가
- 외부 API 호출이 과도하지 않은가

### 6. 코드 구조

- Controller, Service, Repository 책임이 분리되어 있는가
- 비즈니스 로직이 Controller에 몰려 있지 않은가
- 중복 코드가 과도하지 않은가
- 함수/변수명이 의도를 잘 드러내는가
- 기존 프로젝트 구조와 일관성이 있는가

### 7. 테스트

- 정상 케이스 테스트가 필요한가
- 예외 케이스 테스트가 필요한가
- 권한 테스트가 필요한가
- 회귀 테스트가 필요한 기존 기능은 무엇인가
- 수동 테스트 방법이 명확한가

### 8. 배포 위험

- 환경변수 추가/변경이 있는가
- DB migration이 필요한가
- 운영 데이터에 영향이 있는가
- 롤백이 어려운 변경이 있는가
- 외부 API 키/URL 변경이 필요한가

## 출력 형식

아래 형식으로 리뷰해라.

### 전체 평가

- 승인 가능 / 수정 후 승인 / 재검토 필요 중 하나로 판단한다.
- 이유를 2~3줄로 요약한다.

### 치명적 문제

서비스 장애, 보안 문제, 데이터 정합성 문제, 배포 실패 가능성이 있는 항목을 작성한다.

형식:

- 문제:
- 근거:
- 영향:
- 수정 제안:

### 수정 권장 사항

당장 장애는 아니지만 수정하는 것이 좋은 항목을 작성한다.

형식:

- 문제:
- 이유:
- 개선 제안:

### 사소한 개선

네이밍, 가독성, 중복 제거 등 작은 개선 사항을 작성한다.

형식:

- 위치:
- 제안:

### 테스트 보완

추가로 검증해야 할 테스트를 작성한다.

- 정상 케이스:
- 예외 케이스:
- 권한 케이스:
- 회귀 테스트:

### 최종 판단

- 승인 가능
- 수정 후 승인
- 재검토 필요

---

## 리뷰 원칙

- 근거 없는 취향 지적은 하지 않는다.
- 실제 장애 가능성이 높은 문제를 우선한다.
- 코드 스타일보다 요구사항, 안정성, 보안, 데이터 정합성을 우선한다.
- 수정 제안은 구체적으로 작성한다.
- 불확실한 부분은 “확인 필요”라고 표시한다.
- 기존 코드 컨벤션을 우선한다.
- 대규모 리팩토링을 무리하게 요구하지 않는다.
